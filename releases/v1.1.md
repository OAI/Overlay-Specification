# Overlay 1.1 Released!

## New features

- New copy action that can be used to copy/move an element in the OpenAPI document.
- New `description` field for the Overlay info object.
- Better update/delete capabilities for primitive values.
- [RFC9535](https://tools.ietf.org/html/rfc9535) compliance for better interoperability.
- File naming convention for better tools integration.
- More details on the different formats supported by the specification.

More information on the [upgrade documentation](https://learn.openapis.org/upgrading/overlay-v1.0-to-v1.1.html).
